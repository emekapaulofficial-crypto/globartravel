        # Globar Travel Agency — Website

Responsive travel agency website for **Globar Travel Agency** (Nigeria), built with HTML5, CSS3, and vanilla JavaScript.

## Features
- Flight booking, visa assistance, holiday packages & study-abroad/scholarship guidance
- Dark / light mode toggle
- Smooth-scroll navigation with active-link highlighting
- Animated stats (split-flap style counters)
- Destination showcase & tour packages
- Testimonial slider
- FAQ accordion
- WhatsApp-integrated contact form

## Tech Stack
- HTML5 / CSS3 / JavaScript (no frameworks)
- [Font Awesome](https://fontawesome.com/) for icons
- [AOS](https://michalsnik.github.io/aos/) for scroll animations
- Google Fonts (Fraunces, Inter, Space Mono)

## Files
- `index.html` — page structure
- `style.css` — styling & theme
- `script.js` — interactivity (nav, dark mode, slider, form)

## Usage
Clone or download the repo and open `index.html` in a browser — no build step required.
